imgspider
=========

图片爬虫：采用多线程图片下载和写入，速度较快，便于扩展`

使用说明
-------------------------
详见 tese.java
输入的image.txt 为图片的下载地址列表，如下：
*
*http://113.107.42.72:88/images/uploadimg/20096322571612461.jpg
*http://113.107.42.72:88/images/uploadimg/20094281529157735.jpg
